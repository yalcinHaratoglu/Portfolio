- **Raporlama & Dokümantasyon:** `@react-pdf/renderer` ile çok adımlı dinamik PDF rapor sihirbazı; 30+ bileşen için Storybook 8 (~60 story) ile design system dokümantasyonu.
- **Test & CI:** Playwright ve Cypress ile kritik kullanıcı yolları (CRUD, Auth) için E2E testleri; GitHub Actions CI entegrasyonu.

## Bu Proje Hakkında

Ratio EMS, RatioSIM ve ilgili modüllerin vitrinlendiği **çok dilli (TR/EN)** portföy sitesi.

### Öne çıkan özellikler

- Hero, Hakkımda, Yetenekler, Projeler ve İletişim bölümleri
- 10 proje için detay sayfaları (`/projects/[slug]`)
- TR/EN i18n, dark/light tema
- Mobil alt dock navigasyon
- **Benimle Çalışın** iletişim formu (Nodemailer + Gmail SMTP)
- CV indirme (EN/TR PDF)
- Vercel Analytics & Speed Insights

### Tech stack

Next.js 15 · React 19 · TypeScript · Tailwind CSS · i18next · react-hook-form · Zod · Nodemailer · pnpm

### Kurulum

```bash
pnpm install
cp .env.example .env.local   # SMTP ve iletişim ayarları
pnpm dev
```

**Ortam değişkenleri** (`.env.local` / Vercel):

```env
SMTP_USER=myalcinharatoglu@gmail.com
SMTP_PASS=your_gmail_app_password
CONTACT_TO_EMAIL=myalcinharatoglu@gmail.com
CONTACT_FROM_EMAIL=myalcinharatoglu@gmail.com
```

**Statik dosyalar:**

```
public/cv/cv-en.pdf
public/cv/cv-tr.pdf
public/projects/{slug}.jpg
public/profile.png
```
